Staging area gives you another layer of flexibility so that you can choose which files in the directory to track.

Add only the file with changes that you want to commit to the staging area so that you can make one commit per logical change.

Git branch allows you to make experimental changes and start side project without affecting the current work.

Diagram illustrates the topology of the branch structure.

Automatic merging makes merging faster and easier. On the other hand it may do it incorrectly.

'fork' is cloning repository on the github server. 'clone' copies the repository onto your local computer. 'branch' makes a new course of development of the project.
